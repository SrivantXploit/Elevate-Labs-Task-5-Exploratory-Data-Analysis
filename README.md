# Task 5: Exploratory Data Analysis (EDA)

## Data Analyst Internship – Elevate Labs

This repository contains the completed **Task 5: Exploratory Data Analysis (EDA)** project. The objective of this task is to extract meaningful insights from the Titanic dataset using statistical analysis and data visualization techniques.

## Dataset

The analysis uses the **Titanic Dataset**, which contains passenger information such as:

- Passenger class
- Gender
- Age
- Fare
- Number of siblings/spouses
- Number of parents/children
- Survival status

## Tools and Libraries Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Exploratory Data Analysis Performed

The following steps were performed:

- Dataset inspection and understanding
- Missing value analysis and handling
- Summary statistics
- Age distribution analysis
- Fare distribution and outlier detection
- Survival analysis by gender
- Survival analysis by passenger class
- Age vs Fare analysis
- Correlation analysis using a heatmap
- Pairplot analysis
- Extraction of key insights and observations

## Key Insights

- The overall survival rate was **38.38%**.
- Female passengers had a much higher survival rate (**74.20%**) than male passengers (**18.89%**).
- First-class passengers had the highest survival rate (**62.96%**).
- Third-class passengers had the lowest survival rate (**24.24%**).
- Most passengers were between **20 and 40 years old**.
- Passenger fares contained several high-value outliers.
- Higher fares showed a positive relationship with survival.
- Gender and passenger class were the strongest factors associated with survival.

## Conclusion

This exploratory data analysis identified important patterns in the Titanic dataset. Gender and passenger class had the strongest association with survival, while fare also showed a positive relationship with survival. The analysis demonstrates the use of statistical summaries and visualizations to extract meaningful insights from data.
